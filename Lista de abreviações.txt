########## LISTA DE ABREVIACOES ##########
- ADD = Adicionar !Adiciona um valor qualquer
- AVR = Average = Média
- CAL = Calcula
- CHK = Check = Verificação
- CON = Conveyor = Esteira
- CUR = Current = Atual
- DEC = Decrementa
- DIS = Distance = Distância
- INC = Incrementa
- QTD = Quantidade
- PAR = Parâmetro
- PCA = Peca
- PER = Permissivel
- POS = Posição
- PRE = Presence = Presença !Usado para peças, ferramentas, ou qualquer outro item
- PROC = Process = Processo
- PROD = Produto
- PTP = Ponto
- RES = Reset / Limpa / Retorna ao valor original
- SET = Setup = Atribuí/Configura
- STA = Station = Estação
- STU = Status
- TON = Turn ON = Aciona determinado atuador, ou componente
- TOF = Turn OFF = Desaciona determinado atuador, ou componente
- UPT = Update = Atualiza
